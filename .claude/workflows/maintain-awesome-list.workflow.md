# Workflow: maintain-awesome-list

Check formatting, ordering, duplicates, dead links, and consistency.

## Steps

1. Confirm `README.md` is the only content artefact and no forbidden content directories exist.
2. Check section structure and order match the established list.
3. Check alphabetical order within sections where practical.
4. Check entry format: `- [Name](https://link) - Description.` with neutral, one-line descriptions.
5. Scan for duplicate entries and duplicate links.
6. Run the markdown link check for dead links; fix or flag.
7. Run awesome-lint and repository-health; resolve issues.

## Rules

- Make small maintenance edits only.
- Do not change the structure or thesis without explicit instruction.
- Do not commit or push unless explicitly instructed.
